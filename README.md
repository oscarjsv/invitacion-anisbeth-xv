# Invitación XV Años — Anisbeth

Invitación digital de XV años (una sola página, HTML/CSS/JS) con sobre animado,
música de fondo y secciones: portada, información del evento, código de vestimenta,
confirmación por WhatsApp, carta y despedida.

## Uso
Abrir `index.html` en el navegador. Diseñada para móvil.

## Música
El reproductor busca el archivo `perfect.mp3` en la raíz. El incluido es solo un
tono de prueba; reemplázalo por el audio definitivo manteniendo el mismo nombre.
